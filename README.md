## 🚀 Como Executar / How to Run

### Pré-requisitos / Prerequisites
* Node.js (versão 18 ou superior)
* NPM ou Yarn

### Passos / Steps

1. **Instalação das dependências do Frontend:**
   ```bash
   cd frontend
   npm install
   ```

2. **Executar em modo de desenvolvimento:**
   ```bash
   npm run dev
   ```

3. **Gerar build de produção:**
   ```bash
   npm run build
   ```
